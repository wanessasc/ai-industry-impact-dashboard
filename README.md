🇧🇷 Sobre o Projeto

Projeto de análise de dados desenvolvido em Python para explorar o impacto da adoção de Inteligência Artificial em diferentes indústrias e países.

O projeto inclui:

* Pipeline completo de dados
* Análise exploratória (EDA)
* Dashboard interativo desenvolvido com Streamlit

Tech Stack:

* Python
* Pandas
* NumPy
* Plotly
* Streamlit
* Matplotlib
* Seaborn

Estrutura do Projeto:

ai-industry-impact-dashboard/
├── data/
│   ├── raw/
│   └── processed/
├── notebooks/
├── src/
├── dashboard/
└── requirements.txt

Como Rodar corretamente:

Instale as dependências:
pip install -r requirements.txt

Rode o pipeline:
python -m src.pipeline

Rode o dashboard:
streamlit run dashboard/app.py

Funcionalidades do Dashboard:

* Métricas de adoção de IA
* Análise do impacto na receita
* Análise do impacto no emprego
* Pontuação de maturidade da IA
* Comparações entre países e setores
